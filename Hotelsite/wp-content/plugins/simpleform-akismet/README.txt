Do you get junk emails through your contact form? This plugin for SimpleForm helps prevent spam submission.

== Description ==

A lightweight addon for SimpleForm plugin. No configuration required. Once activated, you will find the new “Akismet Anti-Spam” option in the Settings page within the Spam Protection tab. Ensure that before you enable this option, you have activated Akismet plugin and registered an Akismet API Key for enabling anti-spam protection. You no longer need to worry about getting spam in your contact form, since each new form submission will be filtered for spam using Akismet.

For more detailed information refer to the [SimpleForm](https://wordpress.org/plugins/simpleform/) plugin page.

== Installation ==

Activating the SimpleForm Akismet plugin is just like any other plugin.

= Using the WordPress Dashboard =

1. Navigate to “Add New” in the plugins dashboard
2. Search for “SimpleForm Akismet”
3. Click “Install Now”
4. Activate the plugin in the Plugin dashboard

= Uploading into WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select “simpleform-akismet.zip” in your computer
4. Click “Install Now”
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download “simpleform-akismet.zip”
2. Extract the “simpleform-akismet” directory in your computer
3. Upload the “simpleform-akismet” directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard

== Frequently Asked Questions ==

= Can I install this plugin without SimpleForm? =

No. You cannot. This is an addon plugin and it requires SimpleForm plugin to work.

= Why is this feature not integrated into the SimpleForm plugin? =

We have designed SimpleForm to be a minimal, lightweight, fast and privacy-respectful plugin, and we want to keep it that way. You can choose to enable this feature at any time, and with a single click, depending on your needs.
